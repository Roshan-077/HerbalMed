# HerbalMed

## Description
HerbalMed is a web application dedicated to medicinal plants and their uses. It provides information about various plants, their properties, and traditional uses.

## Starting File
The starting file for the application is `index.html`.

## Dependencies
- Bootstrap CSS and JS (from CDN) for styling and responsive design.
- Custom styles located in `styles/style.css`.

## Features
- A navigation bar with links to different sections of the application.
- A carousel for showcasing images of plants.
- Detailed sections for plant profiles, including their medicinal benefits and uses.

## Steps to Follow
1. Clone the repository.
2. Open the `index.html` file in a web browser.
3. Navigate through the application using the provided links.
